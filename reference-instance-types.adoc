= サポートされているNetApp Data Classificationインスタンスタイプ
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
NetApp Data Classificationソフトウェアは、特定のオペレーティングシステム要件、RAM要件、およびソフトウェア要件を満たすホスト上で動作する必要があります。クラウド上でData Classificationを導入する場合、最大限の機能を発揮させるためには、「大規模」な特性を持つシステムのご利用をお勧めします。

以下の表において、「default」とマークされたシステムがData Classificationをインストールするリージョンで利用できない場合、表に記載されている次のインスタンスタイプがデプロイされます。



== AWSインスタンスタイプ

[cols="20,30,25"]
|===
| システムサイズ | 仕様 | インスタンスタイプ 


| 特大 | 32 個の CPU、128 GB の RAM、1 TiB の gp3 SSD | https://aws.amazon.com/ec2/instance-types/m6i/["m6i.8xlarge"^]（デフォルト） 


| 大規模 | 16 CPU、64 GB RAM、500 GiB SSD | https://aws.amazon.com/ec2/instance-types/m6i/["m6i.4xlarge"^](デフォルト) m6a.4xlarge m5a.4xlarge m5.4xlarge m4.4xlarge 
|===
